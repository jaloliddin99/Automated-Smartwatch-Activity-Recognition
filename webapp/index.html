How to Use

  1. Upload watch/motion-sender.js to your Bangle.js using the Espruino Web IDE
  2. Press the button on the watch to start sending motion data
  3. Click Connect Watch above and select "BangleMotion"
  4. Click Record and tilt your hand to draw
  5. Click Stop when done
  6. Click Export G-code to download the file
  7. Copy the .gcode file to SD card and load on Ultimaker 2+
